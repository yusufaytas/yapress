## How to Create Image Grids

YaPress includes a built-in image grid system that automatically arranges images in responsive column layouts. Here's how to use it:

### Basic Usage (3 Columns - Default)

Wrap your images in a `<div>` with the `className="image-grid"`:

```html
<div className="image-grid">
  <img src="/path/to/image1.jpg" alt="Description 1" />
  <img src="/path/to/image2.jpg" alt="Description 2" />
  <img src="/path/to/image3.jpg" alt="Description 3" />
</div>
```

### Different Column Layouts

You can specify the number of columns by using different class names:

**2 Columns:**
```html
<div className="image-grid-2">
  <img src="/image1.jpg" alt="Image 1" />
  <img src="/image2.jpg" alt="Image 2" />
</div>
```

**3 Columns (default):**
```html
<div className="image-grid">
  <img src="/image1.jpg" alt="Image 1" />
  <img src="/image2.jpg" alt="Image 2" />
  <img src="/image3.jpg" alt="Image 3" />
</div>
```

**4 Columns:**
```html
<div className="image-grid-4">
  <img src="/image1.jpg" alt="Image 1" />
  <img src="/image2.jpg" alt="Image 2" />
  <img src="/image3.jpg" alt="Image 3" />
  <img src="/image4.jpg" alt="Image 4" />
</div>
```

**5 Columns:**
```html
<div className="image-grid-5">
  <img src="/image1.jpg" alt="Image 1" />
  <img src="/image2.jpg" alt="Image 2" />
  <img src="/image3.jpg" alt="Image 3" />
  <img src="/image4.jpg" alt="Image 4" />
  <img src="/image5.jpg" alt="Image 5" />
</div>
```

### Features

- **Responsive Layout**: Column layouts apply on desktop (screens 64rem/1024px and wider)
- **Mobile Friendly**: Automatically adapts to smaller screens with auto-fit columns (minimum 250px per column)
- **Consistent Styling**: All images have rounded corners and borders
- **Equal Heights**: Images use `object-fit: cover` to maintain consistent grid appearance

### Best Practices

1. **Use descriptive alt text** - Always include meaningful alt attributes for accessibility
2. **Optimize image sizes** - Use appropriately sized images (768px-1024px width recommended)
3. **Group related images** - Organize images by theme or location for better storytelling
4. **Add context** - Include descriptive text before each grid to provide context
5. **Choose appropriate columns** - Use 2-3 columns for portraits, 4-5 for thumbnails

### Example with Context

```markdown
## My Trip to Paris

Exploring the city of lights - from the Eiffel Tower to charming cafes, 
Paris offered unforgettable experiences at every corner.

<div className="image-grid">
  <img src="/paris-eiffel.jpg" alt="Eiffel Tower at sunset" />
  <img src="/paris-cafe.jpg" alt="Parisian street cafe" />
  <img src="/paris-louvre.jpg" alt="The Louvre Museum" />
</div>
```

### Technical Details

The image grid uses CSS Grid with the following properties:
- Available classes: `image-grid` (3 col), `image-grid-2`, `image-grid-4`, `image-grid-5`
- Gap: `1rem` between images
- Image styling: `object-fit: cover` with `border-radius: 0.5rem`
- Responsive: Auto-fit on mobile, fixed columns on desktop (64rem+)

This ensures a clean, professional gallery layout without any additional configuration.
