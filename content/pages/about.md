This site is powered by YaPress, a markdown-first publishing engine designed for static deployment and clean ownership boundaries.

## Our Mission

We believe content should be portable, version-controlled, and owned by its creators. YaPress makes that possible by treating markdown files as the source of truth, not a database or proprietary format.

## What Makes YaPress Different

### Content Ownership

Your content lives in plain markdown files. No vendor lock-in, no export hassles. Just files you can edit with any text editor and version control with Git.

### Framework/Site Separation

YaPress maintains a strict boundary between framework code and your content. This means:

- **Predictable upgrades**: Framework updates don't break your content
- **Portability**: Move your content to another system anytime
- **Flexibility**: Run multiple independent sites on the same engine

### Built for Developers

If you're comfortable with Git, markdown, and the command line, you'll feel at home with YaPress. No admin panels, no databases, no complexity.

## Technology Stack

- **Next.js 15**: Modern React framework with App Router
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Utility-first styling
- **MDX**: Markdown with React components
- **Static Export**: Pure HTML/CSS/JS output

## Open Source

YaPress is open source and free to use. Contributions, feedback, and bug reports are welcome on [GitHub](https://github.com/yapress/yapress).
